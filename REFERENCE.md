# Language Reference

## Variables
```
let int_var = 10;
let string_var = "Hello World";
```

## Printing to standard output
```
let var = "Hello, World!";
stdout(var);
```

## If statements
```
let x = 7;
if (x != 8) {
    stdout(90);
}
```

## While loops
```
let foo = 0;
while (foo != 100) {
    stdout(foo);
    foo = foo + 1;
}
```

## Lists
```
let lst = [1, 2, 3, 4, 5];
let i = 0;
while (i != 5) {
    stdout(lst[i]);
    i = i + 1;
}
```
